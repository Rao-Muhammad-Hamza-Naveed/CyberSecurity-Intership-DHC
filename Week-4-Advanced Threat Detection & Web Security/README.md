\# Week 4 – Advanced Threat Detection \& Web Security



\## Overview

This project implements advanced web security controls aligned with OWASP Top 10

using Node.js and Express.



\## Features Implemented

\- Intrusion Detection using Fail2Ban

\- API Rate Limiting

\- Secure CORS Configuration

\- API Key Authentication

\- Content Security Policy (CSP)

\- HTTPS Enforcement using HSTS



\## Technologies Used

\- Node.js

\- Express.js

\- Helmet

\- Fail2Ban

\- OWASP Juice Shop (Reference)



\## How to Run

1\. Install dependencies:

&nbsp;  npm install

2\. Start server:

&nbsp;  node server.js



\## API Endpoints

\- `/api/public` – Public access

\- `/api/secure` – Requires API Key



\## Security Benefits

\- Prevents brute-force attacks

\- Protects against XSS

\- Secures API endpoints

\- Detects intrusion attempts
