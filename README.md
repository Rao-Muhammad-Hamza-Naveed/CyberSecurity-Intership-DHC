# Cybersecurity Internship – DevelopersHubs Corporation (DHC)

This repository contains the complete work for a **6-week Cybersecurity Internship** completed at **DevelopersHubs Corporation (DHC)**. Each week’s tasks are organized into separate folders with reports, screenshots, scripts, and other relevant files.

---

## Repository Structure

│
├── Week-1-Security Assessment

├── Week-4-Advanced Threat Detection & Web Security
├── Week-5-Ethical Hacking & Exploiting Vulnerabilities

### Each week folder contains:
- Task descriptions and instructions (`README.md` or `.txt`)
- Scripts and code files where applicable
- Screenshots, reports, and logs of completed tasks

---

## Internship Overview

During this internship, the following areas of cybersecurity were covered:

- **Week 1 – Security Assessment:** Initial security assessment and environment setup.
- **Week 2 – Security Measures:** Implementation of basic security practices and web security.
- **Week 3 – Advanced Security:** Advanced security practices including monitoring, logging, and vulnerability analysis.
- **Week 4 – Advanced Threat Detection & Web Security:** Real-time threat detection, API hardening, and security headers.
- **Week 5 – Ethical Hacking & Exploiting Vulnerabilities:** Controlled penetration testing and ethical hacking exercises.
- **Week 6 – Advanced Security Audits & Final Deployment:** Security audits, system hardening, and preparation for secure deployment.

Each week’s folder contains detailed reports, screenshots, and artifacts to demonstrate task completion.

## Technologies & Tools Used

- **Operating Systems:** Kali Linux, Windows 10

- **Web Security & APIs:** Node.js, Express.js
